# Movie Review Web Application

A full-stack Movie Review Web Application built using Django and Django REST Framework. The application allows users to browse movie listings, view detailed movie information, and interact with movie data through RESTful APIs.

## Features

* Movie listing page with responsive UI
* Movie detail page using slug-based URLs
* Django REST Framework APIs
* API Pagination
* Custom Filtering using django-filter
* MySQL Database Integration
* Dynamic movie data management
* Admin Panel for content management
* Clean and scalable project structure

## Tech Stack

### Backend

* Python
* Django
* Django REST Framework (DRF)

### Database

* MySQL

### Frontend

* HTML
* CSS

## API Features

* Retrieve all movies
* Retrieve movie details
* Page-wise API responses
* Filter movies by year and other fields
* RESTful API architecture
